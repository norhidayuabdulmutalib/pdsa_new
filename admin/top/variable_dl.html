Information

This one was requested by G. Wolfgang so here it is, a drop down definition list that has variable width top level and sub level links. This took longer than I thought to produce but I got there in the end :)

Again it works in IE6, IE7, Opera 8.5, Opera 9, Firefox 2 etc. IE5.x will see the top level links only.
